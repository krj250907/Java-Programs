# Java Programs

A collection of 10 basic Java programs for beginners.

## Programs Included

1. Hello World
2. Calculator
3. Prime Number Checker
4. Fibonacci Series
5. Factorial
6. Palindrome Number
7. Armstrong Number
8. Star Pattern
9. Largest Element in Array
10. Reverse String

## Technologies Used

- Java
- Visual Studio Code
- Git
- GitHub

## How to Run

1. Open the project in Visual Studio Code.
2. Open any Java file from the `src` folder.
3. Click **Run** or execute:

```bash
javac FileName.java
java FileName
```
